MeiliSearchClient

public class MeiliSearchClient

A MeiliSearchClient instance represents a MeiliSearch client used to easily integrate your Swift product with the MeiliSearch server.

Warning

MeiliSearchClient instances are thread safe and can be shared across threads or dispatch queues.

    Add a list of documents and update them if they already.

    If you send an already existing document (same id) the old document will be only partially updated according to the fields of the new document. Thus, any fields not present in the new document are kept and remained unchanged.
